<h1>0.사용기술</h1>
<P>- JAVA 17
<P>- Spring Reactive Web
<P>- Maven
<P>- Lombok
<P>- Reactive mongoDB

<h1>1.프로젝트 목표</h1>
<P>- 비동기 Netty 서버 만들어보기
<P>- NoSQL MongoDB 사용해보기
<P>- Spring boot 활용 및 어노테이션 사용해보기
<P>- AJAX,JS 공부

<h1>2.구현기능</h1>
<p>- 실시간 채팅 기능
<p>- 로그인 시스템 대신 아이디와 채팅방번호 입력 입장

<h1>3.추후 추가 기능</h1>
